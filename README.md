# XDesktop (React Application)

XDesktop is a React-based desktop environment simulation with draggable windows, folders, taskbar tabs, context menus, and a virtual filesystem.

## Features

- Desktop icons
- Folder windows
- Taskbar with window tabs
- Start menu
- Drag and drop
- Context menus
- Text file opening in notepad-style windows
- Virtual filesystem model

## Tech Stack

- React
- JavaScript
- CSS
- Vite

## Getting Started

```bash
npm install
npm run dev
