# XDesktop (React Web Application)

XDesktop is a desktop-style web environment built with React.  
It simulates a traditional operating system desktop with icons, windows, folders, taskbar tabs, and a virtual filesystem running entirely inside the browser.

---

## Live Demo

View the application running on GitHub Pages:

**https://dgiannop.github.io/xdesktop-react/**

---

## Features

- Desktop icons
- Folder windows
- Window management (move, focus, minimize)
- Taskbar with window tabs
- Start menu
- Drag & drop support
- Context menus
- Text files opening in Notepad-style windows
- Virtual filesystem model
- Desktop-style UI layout

---

## Tech Stack

- **React**
- **JavaScript (ES Modules)**
- **CSS**
- **Vite**
